Let A, B, C, X be arrays.<br>

A, B : n-element arrays. <br>
X : n-element array which is used for carry. <br>
C : (n+1)-element array which stores result of A + B. <br><br>


First, X[i] = 0 for all 0 =< i =< n <br>

for j=n to 1
  if A[j] + B[j] + X[j] >= 2
    X[j-1] = 1 
    C[j]= (A[j] + B[j] + X[j]) %2
  else
    C[j] = A[j] + B[j] + X[j]

C[0] = X[0]
